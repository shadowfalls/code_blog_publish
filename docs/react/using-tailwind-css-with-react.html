The following tutorial is for React apps that are built with Create React App v5.0+. If you are looking to add Tailwind for React apps built with Create React App with v4.0 or lower you can refer the code in this repo

In this tutorial we will learn what Tailwind CSS is and how to use it with ReactJS. You can imagin Tailwind css as huge set of utility classes that you can use globally everywhere in your application.

Why use Tailwind css:

Traditional css don't work as your app gets bigger you will keep adding more css. This leaves a lot of dead css in the application, this is code which may or may not be needed or executed in browser. When every you need to add a new class, you have to think hard for a proper/unique class name.

Tailwind CSS is a utility first CSS framework and has classes for any css property like flex, pt-4, text-center and rotate-90 that can be composed to build any design, directly in your markup. Tailwind is intelligent enough to know which classes are not being used so that the final build file does not have css you did use.

Why not use Tailwind css:

Given all the benifits there are a few things you need to know before adding it to your project.

  • If you are not comfortable with writing html tags with many classes
  • You need to get used to a new way of writting CSS
  • The code will get difficult to read
  • It is hard to tweak CSS in developer tools
  • It Makes PRs Harder to Review

Add Tailwind css to React:

We will be concentrating on React apps that are build with Create React app. If you are not familier with React we recommend getting familier with it from their documentation.

1. Start by creating a new React project if you don't have a project already setup up.

2. Install tailwindcss and its peer dependencies via npm, and then run the init command to generate both tailwind.config.js and postcss.config.js.

3. Add the paths to all of your template files in your tailwind.config.js file.

4. Add the @tailwind directives for each of Tailwind's layers to your ./src/index.css file or the file that hosts the global css.

That is all the setup you need to get stated with tailwindcss. If you are using VS code as you code editor, we recommend using this VS code extension that give you support for IntelliSense for Tailwindcss.
